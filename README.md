# NetworkApp
Simple network app

Приложение содержит в себе информацию о героях и эпизодах сериала **"Breaking Bad"**

Данные берутся из сети с публичного api

Приложение содержит:
- MVC
- CollectionView
- TableView
- URLSession
- Alamofire
- Singleton (использутеся для сетевого слоя)
- верстка кодом
